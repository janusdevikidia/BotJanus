# BotJanus
### Description

> [!NOTE]
> BotJanus est un robot développé par Janus qui permet de faire des tâches de maintenance sur les wikis.

### Scripts
Les scripts de BotJanus :

* __```AWBmini.js```__ : Script Javascript à installer sur le ```common.js``` du robot qui permet de faire des actions de maintenance directement en étant connecté sur le wiki via le navigateur.
* __```interwiki-en.py```__ : Permet de mettre des liens interwikis vers [Wikipédia en anglais](https://en.wikipedia.org).
* __```cmessager-de-masse.py```__ : Permet de distribuer la Gazette de Vikidia.
* __```orphan.py```__ :  Permet d'ajouter le bandeau orphelin en fonctions si la page est orpheline ou non.
* __```uncategorized.py```__ : Permet d'ajouter le bandeau de catégorie manquante en fontion si la page contient des catégories ou non.
* __```welcome.py```__ :  Permet d'ajode bienvenuter les nouveaux arrivants.
* __```flask_app.py```__ : Code source du dashboard de BotJanus (mis à jour pas souvent)


### Informations utiles
> [!NOTE]
> Certains scrits peuvent être lancés via le [Dashboard de BotJanus](https://janus.pythonanywhere.com)
